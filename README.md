## ✨ Architectural Highlights & Features

- ⚡ **Lightning-Fast Core**: Powered by **Vite 8** for instantaneous module replacement and sub-second production builds.
- 🧲 **Cuberto Magnetic Physics**: Custom mathematical interpolation (`Magnetic` component) providing smooth cursor-pull interactions on interactive links and action buttons.
- 🎭 **3D Holographic Parallax**: Engineered a custom 3D card tilt effect (`ProfileCard3D`) calculating exact mouse velocity and perspective coordinates (`translateZ`, `rotateX/Y`).
- 🌊 **Staggered Scroll Reveals**: Intersection Observer API implementation rendering smooth entrance animations as elements enter the viewport.
- 🌓 **Dynamic Theme Engine**: Flawless Dark / Light mode transitions with inverted difference blend cursors ensuring pristine contrast.
- 💬 **Direct Lead Generation**: Floating interactive chat widget connecting visitors directly to automated WhatsApp lead channels and Formspree API endpoints.

## 🚀 Local Development & Setup

To explore or modify the codebase locally, follow these clean architectural steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18+) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/latrecheyasser1-hue/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```
   Open your browser and visit `http://localhost:5173/`.

4. **Build for Production:**
   ```bash
   npm run build
   ```
   This generates an optimized, minified production bundle inside the `/dist` directory.

## 📂 Code Organization & Clean Code Standards

The codebase adheres strictly to clean architecture principles:
- **Separation of Concerns**: Reusable UI wrappers (`<Magnetic>`, `<ScrollReveal>`, `<TextReveal>`) are encapsulated to keep view components clean.
- **Defensive Programming**: Image loaders feature fallback handlers (`onError`) to ensure UI integrity even under network failures.
- **Responsive by Design**: Mobile-first breakpoints (`sm:`, `md:`, `lg:`) guaranteeing flawless rendering on desktop ERP screens, tablets, and smartphones.
